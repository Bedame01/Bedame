# Bedame Portfolio

A Next.js portfolio inspired by the [Kajo Framer template](https://kajo-template.framer.website/) by Pawel Gola.

## Features

- Home, Work, Service, About, Blog, Contact pages
- Light / dark mode
- Framer Motion scroll animations
- Fully responsive layout
- Project and blog detail pages

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize

- **Site info**: `src/data/site.ts`
- **Projects**: `src/data/projects.ts`
- **Blog posts**: `src/data/blog.ts`
- **Team**: `src/data/team.ts`
- **Services**: `src/data/services.ts`

## Deploy

Deploy to [Vercel](https://vercel.com) or any Next.js-compatible host:

```bash
npm run build
```
