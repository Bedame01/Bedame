# Bedame Portfolio

This repository contains the official portfolio website for Bedame Visuals, a creative studio based in Lagos, Nigeria. The site is built to showcase work, services, personality, and contact details in a clean, modern, and responsive experience.

## Overview

The project is a polished personal portfolio website built with Next.js, React, Tailwind CSS, and Framer Motion. It includes dedicated pages for home, work, services, about, and contact, with a dark/light theme and smooth animated transitions.

## Features

- Modern one-page and multi-page portfolio experience
- Responsive layout for desktop, tablet, and mobile
- Smooth scroll and motion-based UI interactions
- Dedicated sections for projects, services, testimonials, and social links
- Custom content driven by structured data files
- Contact form setup for direct inquiries

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- nodemailer for contact handling

## Getting Started

1. Install dependencies
   ```bash
   npm install
   ```

2. Start the development server
   ```bash
   npm run dev
   ```

3. Open the site in your browser at
   ```text
   http://localhost:3000
   ```

## Project Structure

- src/app — route-based pages and layouts
- src/components — reusable UI and section components
- src/data — portfolio content such as site info, projects, services, and testimonials
- src/lib — utility helpers
- public — static assets and images
